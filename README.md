# Advent of code 2019

To run with a certain input, drop it into an input.txt file in a day's folder and run `python solution.py`. Debug statements and some output designed to work with Python 3.7+ due to fstrings.

## Misc

```bash
 function aoc {
   mkdir "${1}"
   touch "${1}/solution.py"
   touch "${1}/input.txt"
 }
 ```
